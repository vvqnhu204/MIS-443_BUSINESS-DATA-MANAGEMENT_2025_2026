# MIS-443_BUSINESS-DATA-MANAGEMENT_2025_2026


## 📌 Project Description
This repository contains the individual assignment (Assignment 4) to recreate the `School` database schema from 
[SQL Practice Online](https://www.sql-practice.online/practice/school?engine=postgresql) using PostgreSQL. 
The project encompasses building the database schema, defining relationships, importing data from CSV files, and solving 30 practical SQL queries.

## 🛠 Tools Used
* PostgreSQL & pgAdmin 4
* SQL Practice Online
* CSV Data Files
* GitHub
